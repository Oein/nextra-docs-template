# String (문자열)

`사과`, `An apple`같은 것들이 문자열 입니다.

## 쓰는 방법

Typescript에서는 문자열을 쓰는 방법이 여러가지에요

```ts
< "이건 문자열 이에요";
> "이건 문자열 이에요"
```

```ts
< '이것도 문자열 이에요';
> "이것도 문자열 이에요"
```

## 다음줄로!

```
사과는 맛있다.
오렌지도 맜있다.
```

를 그대로

```ts
"사과는 맛있다.
오렌지도 맜있다."
```

라고 쓰면 에러가 나요. 하지만 우리에겐 `\n`가 있어요! `\n`은 엔터와 같은 효과를 내요. 그래서 아래와 같이 쓸 수 있어요.

```ts
< console.log("사과는 맛있다.\n오렌지도 맜있다.");
> 사과는 맜있다.
오렌지도 맜있다.
```

## 더 쉬운 다음줄은 없나요?

`\n`을 쓰는건 코드가 가로로 너무 길어지네요.. 그래서 우리는 \`라는걸 사용해 볼꺼에요.\`는 우리가 쓴걸 토시하나 안틀리고 똑같이 써줘요.

```ts
< console.log(`사과는 맜있다.
오렌지도 맜있다.`)
> 사과는 맜있다.
오렌지도 맜있다.
```

## 어라? \\는 어떻게 쓰나요?

`\`를 출력했더니 에러가 나네요..?

```ts
< console.log("\")
> Error / Uncaught SyntaxError: Invalid or unexpected token
```

원래 `\`는 탈출 문자에요. 원래 문자로는 표현되지 않는 기능을 수행하기 위해 있는 특수한 문자이죠. 하지만 일반 텍스트로 쓰고 싶을때는 `\\`로 \\을 두번 쓰면됩니다!

```ts
< console.log("\\가 써지네요!")
> \가 써지네요!
```
