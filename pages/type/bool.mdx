# Boolean (참과 거짓)

Boolean은 참(true)과 거짓(flase)를 나타낼 수 있어요.

```ts
true;
false;
```

조건이 성립하면 true를 아니면 false를 나타내죠.

```ts
if (3 < 5) { // 3 < 5가 true라서 아래의 코드가 실행되는거에요.
    console.log("3 < 5 is true"); // True일때 실행되요.
} else {
    console.log("3 < 5 is false"); // False일때 실행되요.
}

> 3 < 5 is true
```
