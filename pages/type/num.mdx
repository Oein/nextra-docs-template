# Number (숫자)

1, 3.14, -2 같은 숫자는 Number라고 합니다.

## 숫자 연산

우리가 수학을 하는것 처럼 쓰면 사칙연산을 쓸 수 있습니다. 수학에서는 곱셈기호를 `x` 라고 쓰지만 컴퓨터의 세계 에서는 `*` 이라고 쓰고 나누기를 `/` 라고 씁니다.

```ts
< console.log(1 + 1);
> 2
< console.log(1 - 3);
> -2
< console.log(1 * 19);
> 19
< console.log(8 / 4);
> 2
```

## 문자열에서 숫자로

Typescript에는 `parseInt`라는 함수가 내장되있습니다. parseInt에 인자로 String(문자열)을 넘겨주면 됩니다.

```ts
< console.log(parseInt("13.33"))
> 13.33 // Number에요!
```

## 숫자에서 문자열로

Typescript의 Number는 `toString` 이라는 함수를 내장하고 있습니다.

```ts
< let a: Number = 3.14 // 지금은 변수선언을 몰라도 되요.
< console.log(a.toString()) // a는 3.14라는 값을 가지고 있어요.
> "3.14"
```
