# Array\<any\> (모두 다 함께)

Array는 수학의 수열같은 것 이에요. 한곳에 여러 값들을 저장할 수 있어요. 예를 들어 1,2,3을 한곳에 모을려면 아래와 같이 쓸 수 있어요.

```ts
[1, 2, 3];
```

## 변수와 함께하면 더욱 좋아요!

우선 변수를 모른다면 변수를 배우고 오는걸 추천드려요.

Array변수는 값을 조작할 수 있어요.

### 특정 위치 조작하기

```ts
let arr: Array<Number> = [1, 2, 3, 4, 5];

console.log(arr[1]);
> 2
```

```ts
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr[1] = 100;
console.log(arr[1]);
> 100
```

Array[인덱스] 로 Array의 값에 접근이 가능해요. 인덱스는 번호인데 0부터 시작해요.

### 맨 앞을 삭제하기

shift라는 함수로 맨 앞을 삭제할 수 있어요.

```
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr.shift();
console.log(arr);

> [2, 3, 4, 5];
```

### 맨 뒤를 삭제하기

pop이라는 함수로 맨 뒤의 원소를 삭제할 수 있어요.

```
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr.pop();
console.log(arr);

> [1, 2, 3, 4];
```

### 맨 뒤에 추가하기

push라는 함수로 맨 뒤에 원소를 추가할 수 있어요.

```ts
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr.push(6);
console.log(arr);

> [1, 2, 3, 4, 5, 6];
```
