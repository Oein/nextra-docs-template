# 타입

타입의 세계의 오신것을 환영해요! 이 섹션에서는 Typescript의 주요기능인 Type을 배울 수 있어요.

## 무엇을 배우나요?

- Number
- String
- boolean
- any
- null과 undefined
- Array\<any\>
- || 연산자
- Type casting

## 타입이 무엇인가요?

타입은 말그대로 종류에요. 숫자면 Number, 문자열이면 String같이 어떻게 생긴 친구인지 알려주는 표시에요.

## 타입이 왜 있나요?

우리는 3.14 라고 하면 숫자일꺼라고 생각하지만 컴퓨터는 몰라요. 그렇기 때문에 우리가 타입을 알려주어야 해요.
