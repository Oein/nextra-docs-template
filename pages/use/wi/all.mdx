# 개발환경 설정하기

이 파트를 진행하기전에 Node.js와 Yarn을 설치해야해요. Node.js와 Yarn을 설치하지 않았으면 자신에 OS에 맞게 설치해봐요. 강의 목록에 있는 자신에 OS맞는 설치방법 강의를 보고 설치해봐요.

## 설정하기

1. Typescript프로젝트를 설치할 폴더를 만들어요.

2. 터미널에 cd 커맨드를 이용해서 우리가 만든 폴더로 이동해요.

- Windows 예시

```bat
cd C:\Users\Oein\Documents\TypescriptProject
```

- Mac 예시

```sh
cd /Users/Oein/Documents/TypescriptProject
```

3. Yarn을 이용해서 프로젝트를 초기화 해요.

```sh
yarn init -y
```

4. Typescript를 설치해요.

```sh
yarn add typescript
```

5. Typescript를 초기화 해요.

```sh
npx tsc --init
```

6. tsconfig.json을 수정해요.

`// "rootDir": "./",`을 `"rootDir": "./src",`로 수정해요.

`// "outDir": "./",`을 `"outDir": "./dist",`로 수정해요.

7. `src` 폴더를 만들어요.

8. `src`폴더 안에 index.ts를 만들어요.

9. `index.ts`에 아래의 코드를 작성해봐요.

```ts
console.log("Hello, Typescript!");
```

## 실행하기

1. Typescript를 빌드해요.

```sh
npx tsc
```

2. 빌드된 js파일을 실행해요

```sh
node dist/index.js

> Hello, Typescript!
```
