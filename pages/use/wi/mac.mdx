# Mac에 Typescript 계발 환경 설치하기.

## Homebrew 설치하기

이미 Homebrew를 설치했다면 다음단계로 넘어가도 좋아요.

자신이 쓰는 터미널에 아래의 커맨드를 실행하여 Homebrew를 설치해봐요.

```sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Node.js 설치하기

이미 Node.js를 설치했다면 다음단계로 넘어가도 좋아요.

자신이 쓰는 터미널에 아래의 커맨드를 실행하여 Javascript 실행 프로그램 Node.js를 설치해봐요.

```sh
brew install node
```

## Yarn 설치하기

이미 Yarn을 설치했다면 다음단계로 넘어가도 좋아요.

자신이 쓰는 터미널에 아래의 커맨드를 실행하여 **메타**가 만든 npm 패키지 관리자 Yarn을 설치해봐요.

```sh
brew install yarn --ignore-dependencies
```
