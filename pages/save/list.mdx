# Array (우리모두 다함께 리스트에서)

리스트는 Array타입을 가지고 있어요. 아직 Array 타입을 모른다면 이전 강의를 참고해 주세요.

## Array를 선언해봐요

Array는 아래의 코드처럼 선언할 수 있어요. 담을 값들은 [] 안에 ,를 사용해 분류해서 넣을 수 있어요.

```ts
let array_name: Array<String> = ["첫번째 값", "두번째", `세번째 값`];
```

## 특정 위치 조작하기

Array는 원하는 워치의 원소를 마음대로 할 수 있어요.

```ts
let arr: Array<Number> = [1, 2, 3, 4, 5];

console.log(arr[1]); // 인덱스 1번째 값을 불러와 출력하는 코드에요.
> 2
```

```ts
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr[1] = 100; // 인덱스 1번째 값을 100으로 설정하는 코드에요.
console.log(arr[1]);
> 100
```

Array[인덱스] 로 Array의 값에 접근이 가능해요. 인덱스는 번호인데 0부터 시작해요.

## 길이를 알아보기

Array는 length라는 값을 항상 가지고 있어요. Array.length로 가지고 있는 원소의 개수를 알 수 있어요.

```ts
let arr: Array<Number> = [1, 2, 3, 4, 5];
console.log(arr.length);

> 5
```

## 맨 앞을 삭제하기

shift라는 함수로 맨 앞을 삭제할 수 있어요.

```
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr.shift(); // 맨 앞을 삭제하는 코드에요
console.log(arr);

> [2, 3, 4, 5];
```

## 맨 뒤를 삭제하기

pop이라는 함수로 맨 뒤의 원소를 삭제할 수 있어요.

```
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr.pop(); // 맨 뒤를 삭제하는 코드에요
console.log(arr);

> [1, 2, 3, 4];
```

## 맨 뒤에 추가하기

push라는 함수로 맨 뒤에 원소를 추가할 수 있어요.

```ts
let arr: Array<Number> = [1, 2, 3, 4, 5];
arr.push(6); // 맨 뒤에 6을 추가하는 코드에요
console.log(arr);

> [1, 2, 3, 4, 5, 6];
```
