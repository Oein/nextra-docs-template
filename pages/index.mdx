## 왜 Typescript를 배울까요?

![This is fine](https://wompampsupport.azureedge.net/fetchimage?siteId=7575&v=2&jpgQuality=100&width=700&url=https%3A%2F%2Fi.kym-cdn.com%2Fphotos%2Fimages%2Fnewsfeed%2F001%2F401%2F347%2F312.jpg)

Javascript는 10일 만에 만든 언어입니다. 그렇기에 C++, Python같은 언어에서는 상상도 할 수 없는

```js
< 1 + '1' == '11'
> true
```

엄청난 논리가 통하죠. 서비스의 큰 결함이 Type이였다면, 대형 프로젝트의 경우 위와 같은 버그를 찾기가 힘듭니다. 그렇기 때문에 개발자들은 사전에 위와같은 버그를 해결하고자 Typescript를 만들었습니다.
